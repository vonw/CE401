Plot global temperature anomalies from Berkeley Earth¶

© Von P. Walden, Washington State University

First, download the following dataset from Berkeley Earth:

Land_and_Ocean_LatLong1.nc

This may take a minute…, because the file is 400 MB.

import requests
url = 'http://berkeleyearth.lbl.gov/auto/Global/Gridded/Land_and_Ocean_LatLong1.nc'
r = requests.get(url, allow_redirects=True)
open('Land_and_Ocean_LatLong1.nc').write(r.content)
